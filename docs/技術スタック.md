## 3. 技術選定の理由

### Java + Spring Boot
- エンタープライズで広く使われており、実務スキルに直結
- Spring Data JPA による DB操作の効率化
- 豊富なエコシステムとドキュメント

### React（Vite）
- コンポーネント設計・状態管理の学習に最適
- Vite による高速な開発体験
- Next.jsは今回のスコープ外

### PostgreSQL
- 本格的なRDBの学習ができる
- Spring Data JPA との相性が良い
- Docker での環境構築が容易
