## 2. 選定候補（参考）

### フロントエンド

| 候補 | 特徴 |
|------|------|
| React | コンポーネント設計・状態管理の学習に最適 |
| Vue.js | 学習コストが低く、小規模アプリに向く |
| Vanilla JS | フレームワークなしで基礎力を鍛えられる |

### バックエンド

| 候補 | 特徴 |
|------|------|
| Node.js（Express） | JS統一でフロントと言語を揃えられる |
| Python（FastAPI） | シンプルなAPI実装に向く |
| なし（フロントのみ） | LocalStorage で完結する場合は不要 |

### データ保存

| 候補 | 特徴 |
|------|------|
| LocalStorage | バックエンド不要・ブラウザ完結 |
| SQLite | ローカルファイルDBで軽量 |
| PostgreSQL / MySQL | 本格的なRDB学習ができる |
